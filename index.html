    How To Use

    • If list is empty, add a new list by simply clicking / tapping "Add New List" button.
    • Click on a list, it will bring you to the roullete page.
    • On roullete page, add choices at the box, thus spin the roullete by dragging / swiping the created roullete.
    • Roullete will be drawn if number of choices is more than 0 (zero).
